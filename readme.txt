IntelliDraft is a WordPress plugin that uses AI to streamline content creation, helping users easily produce and optimize high-quality content.


== Description ==

**IntelliDraft – AI-Powered Content Generation for WordPress**

IntelliDraft is a cutting-edge WordPress plugin that leverages the power of AI to revolutionize your content creation process. Whether you're a blogger, marketer, or site administrator, IntelliDraft helps you produce high-quality, engaging content effortlessly. Transform the way you create, manage, and optimize your WordPress site with AI-driven insights and automation.


= Key Features =

* **Blog Post Creation**: Generate complete blog posts with a click. Input your topic or keyword, and let IntelliDraft create content tailored to your specified tone, length, and style.

Experience the future of content creation with IntelliDraft. Transform your WordPress site into a content powerhouse with AI at your fingertips.

**Get started with IntelliDraft today and elevate your content game!**

**Setting Up Your API Key:**

- **API Key Requirement**: To use IntelliDraft, you must create and set up a ChatGPT API key. This is essential for enabling the AI-powered content generation features.
- **How to Set Up**: Go to the WordPress dashboard, navigate to **Settings > IntelliDraft**, enter your ChatGPT API key, save the settings, and select your preferred AI model. Both steps are required for the plugin to function properly.


**Future Features Coming Soon:**

- **Blog Post Featured Image Creation**: Automatically generate and insert relevant featured images into your blog posts.
- **Advanced Content Customization**: Further refine the tone, style, and structure of your AI-generated posts.
- **Additional SEO Tools**: Enhanced keyword integration and optimization features to supercharge your content’s performance.


== Installation ==

1. Upload the `intellidraft` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Navigate to **Settings > IntelliDraft**, enter your ChatGPT API key, save your settings, and select your preferred AI model.
4. Start generating blog posts by navigating to the IntelliDraft section in your WordPress editor.


== Frequently Asked Questions ==

= How does IntelliDraft generate content? =
IntelliDraft uses advanced AI technology powered by GPT-4 to generate content based on the title and topics you provide. The AI is designed to create well-structured, engaging, and SEO-friendly blog posts.

= Do I need an AI API key to use IntelliDraft? =
Yes, you must create a ChatGPT API key and set it up in the plugin settings. Go to **Settings > IntelliDraft** in your WordPress dashboard to enter the API key and select your AI model.

= Can I customize the content generated by IntelliDraft? =
Yes, you can provide specific titles and topics to guide the content generation process. Future versions of the plugin will offer even more customization options, including tone and style adjustments.

= Is IntelliDraft compatible with all WordPress themes and plugins? =
IntelliDraft is designed to work seamlessly with most WordPress themes and plugins. If you encounter any compatibility issues, please contact our support team.
